# Fire-Safety-Planning-Quotation-System

## API Reference

A Swagger-style API guide is available in [SWAGGER_README.md](SWAGGER_README.md).

You can also review the OpenAPI spec in [openapi.yaml](openapi.yaml).

### Quick start

```bash
npm install
npm start
```

Then open:
- http://localhost:5000/ for the health check
- http://localhost:5000/api/equipment for equipment endpoints
